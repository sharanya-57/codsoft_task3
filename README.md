# 🌸 Iris Flower Classification & Prediction App

This project is part of my **Data Science Internship at CodSoft**, where I built a machine learning model to classify iris flowers and deployed it using a Streamlit web application.

---

## 🔍 Project Overview

The goal of this project is to classify iris flowers into three species:

* 🌼 Setosa
* 🌷 Versicolor
* 🌺 Virginica

based on the following features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

---

## ⚙️ Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn
* Streamlit

---

## 📊 Part 1: Classification Model

* Implemented **Logistic Regression**
* Performed **train-test split (80/20)**
* Evaluated model using:

  ✔️ Accuracy Score (~95–100%)
  ✔️ Confusion Matrix

### 📈 Results

* The model achieved high accuracy
* Successfully classified all three flower species
* Confusion matrix shows strong performance

---

## 🌐 Part 2: Prediction Web App (Streamlit)

An interactive web application built using **Streamlit** that allows users to:

* 🎛 Adjust flower measurements using sliders
* 🔍 Predict the species in real-time
* 🌸 Display the corresponding flower image
